## Featured Projects

### [TaskFlow - Task Management System](https://github.com/malisevdinoglu/TaskFlow__SwiftUI)
A comprehensive iOS task management application with advanced features:
- Firebase Authentication & role-based access control
- SwiftData for local data persistence with Firestore sync
- SLA tracking with real-time visual indicators and local notifications
- Digital signature collection and media management
- Automated PDF report generation
- Modern UI with dark theme and card-based design

**Tech**: SwiftUI, SwiftData, Firebase (Firestore, Storage, Auth), UNUserNotificationCenter

---

### [GenAI Recipe Chatbot - RAG Implementation](https://github.com/malisevdinoglu/Akbank-GlobalAIHub_GenAI-Chatbot)
An AI-powered recipe assistant built with RAG (Retrieval-Augmented Generation) architecture:
- Semantic search using Google's text-embedding-005 model
- Natural language query processing with Google Gemini
- ChromaDB vector database for efficient recipe retrieval
- Deployed web application on Render
- Prevents AI hallucinations by grounding responses in actual data

**Tech**: Python, LangChain, Google Gemini, ChromaDB, Streamlit, Pandas

**Live Demo**: [https://genai-chatbot-jc59.onrender.com](https://genai-chatbot-jc59.onrender.com)
